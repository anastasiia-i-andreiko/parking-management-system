#  Parking Management System

Parking Management System — це Full-stack веб-додаток для автоматизації обліку транспортних засобів на паркувальному майданчику. Система дозволяє реєструвати заїзд автомобілів, відстежувати їхній поточний статус у реальному часі, фіксувати час виїзду та керувати базою даних через зручний інтерфейс.

---

##  Як запустити проект

### 1. Підготовка середовища
[cite_start]Створіть файл .env у кореневій папці проекту (він ігнорується гітом згідно з .gitignore та вкажіть шлях до вашої бази даних:
```DATABASE_URL=postgresql://ваш_користувач:пароль@localhost:5432/назва_бд```

### 2. Встановлення залежностей
```pip install -r requirements.txt```

### 3. Запуск Backend-сервера
```python main.py```

### 4. Запуск Frontend
Відкрийте файл index.html у будь-якому сучасному браузері. Переконайтеся, що в app.js змінна API_URL вказує на правильний порт вашого сервера.
```const API_URL = "http://127.0.0.1:8000/api";```

##  Основний функціонал

* Реєстрація заїзду: Додавання транспортного засобу з вказанням номера та типу (легкове, вантажівка, мотоцикл, автобус, велосипед).
* Валідація номерів: Автоматична перевірка формату номера (від 3 до 10 символів, латиниця та цифри).
* Контроль статусів: Можливість позначити виїзд авто (status: left) із автоматичною фіксацією часу виїзду.
* Управління даними: Функція повного видалення записів про транспорт із бази даних.
* Візуальний моніторинг: Список авто з іконками для кожного типу транспорту та відображенням часу заїзду.
